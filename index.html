Famous Cat Facts

JS Cat Facts API Geolocation Web API OpenWeatherMap API Chrome Extension CSS HTML

I have created a Chrome Extension dashboard since I love cats and, I wanted to make something practical while using Chrome for browsing. Hence the idea of putting a few details that I would like to see when I open a new tab, such as interesting cat facts, current time and current weather based on my location.
